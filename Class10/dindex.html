Introduction

This is a brief overview of Digital Shores. It explains what the program is about, how it is installed, and how it functions.

Installation

Follow the steps below to get the project up and running on your local machine:

  1. Clone the repository. Using a program like GitHub Desktop is strongly recommended. git clone https://github.com/your-repository.git
  2. Navigate to the project directory: cd project-directory Alternatively, it can also be run by selecting the Start.exe file in Digital Shores's main folder.
  3. Install dependencies: npm install

Usage

To use this project, simply run the following command:

npm start

Visit http://localhost:3000 in your browser to see the application in action.

Once the application is loaded, install a Large Language Model into the program (Not included).

Examples

Type any message as if you were conversing with another person.

'Hello there!'

Or

'I need directions!'

API

Our project exposes the following API endpoints:

  • GET /api/data - Fetches data from the server.
  • POST /api/data - Submits data to the server.
